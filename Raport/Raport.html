1.Description Of The Site

The site is named Artifacty.

The site allows the user to view information about archaeological artifacts collections, to search for a specific artifact or to view different statistics about collections.

A new user can register or login. An authenticated user can import into the database a new artifact with the permission of the administrator, can subscribe to newsletter and receive an email when a news is posted. He can also post a feedback.

Any user can find out informations about the site and collections.

4.Description of the modules

4.1 Collections

Author: Hostiuc Danut

This page offers the user the posibility to view a slideshow of artifact pictures and informations about collections of archaeological artifacts, structured by types, roles, according to geographic location etc.

4.2 Statistics

Author: Rachieru Larisa

This page offers the user the posibility to view different statistics about collections that matches some criteria.

For example: all mortuary mask artifacts discovered in the 20th century in Africa or those that are not yet dated/researched.

You can also use a text field to fill with some key words for better results. You can just check the boxes and press search and the results will be displayed on the screen.

All those check-boxes creates a query that searches in the database with multiple WHERE clauzes,

4.3 News

Author: Rachieru Larisa

This page contain the latest news about archaeological artifacts.

The user can subscribe to newsletter.

Only the admin can add new information on this section.

4.4 Feedback

Author: Andronic Teodora

Here, users can leave some impressions/feedback about the informations found.

The reviews will be moderated by an admin and after that they will be posted on the page.

4.5 Login

Author: Hostiuc Danuț

This page offers the user the posibility to login so that he can post a new artifact using the import button or to post a feedback.

The login action uses a text field for the email and a password field for the password. When the login button is pressed, the script checks if the email exists in the users database and if the email and password matches.

If the user doesn't have an account, they can use the link to the register page.

4.5.1 Register

This page offers the user the posibility to register so that he can have the priviledges of an logged-in user.

This action will create a new user in the users database.

4.6 Search

Author: Andronic Teodora

This page offers the user the ability to search more quickly an artifact using the text field and the search button, when an artifact's name is written in the text field. The informations will be displayed on the page.

This action creates a query that acceses the database of the artifacts and returns the informaton about the artifact that was searched.

The user can search the artifacts only by they're name

For example: all the masks from the database.

4.7 Content Management System

Author: Elvis Tîncu

This page offers the admin the posibility to add, delete, and modify more quickly a collection in database.

This action creates queries that acceses the database of the artifacts and do the actions:add, delete or modify.
